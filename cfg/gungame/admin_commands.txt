//----------------------------------------------------------------------------
// admin_commands.txt -- Admin command permission definitions.
//
// Each command has a required permission level, 1, 2 or 3. What level each
// admin is are defined in the admins.txt file.
//
// Format: <Command> <Level>
//
// Example: teleport 3
//            Will allow only level 3 admins to use gg_teleport and
//            !ggteleport.
// Example: setlevel 1
//            Will allow level 1, 2 and 3 admins to use gg_setlevel and
//            !ggsetlevel
// 
// Level Info: 0 = Root: this level should be used for really high-level
//                       commands that should only be used by the server owner.
//             1 = Super Admin: can run high-level commands such as
//                              "admin_add".
//             2 = Admin: can run simple command such as "dm_add", etc.
//             3 = All Users: any player can run this command.
// 
// Note: Any edit of this file will require a server restart for changes to
//       take effect.
//----------------------------------------------------------------------------
